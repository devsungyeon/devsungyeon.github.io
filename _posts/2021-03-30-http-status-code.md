# 자주 쓰이는 HTTP status code

궁금한 점이나 오류는 댓글로 달아주시면, 답변 혹은 수정하겠습니다! ":)"


- 100번대 ; 정보 확인
- 200번대 ; 통신 성공
- 300번대 ; 리다이렉트
- 400번대 ; 클라이언트 오류
- 500번대 ; 서버 오류


- 400번대
	- 400 ; Bad Request
	- 401 ; Unauthorized
	- 403 ; Forbidden
	- 404 ; Not found
	- 405 ; Method Not Allowed
	- 406 ; Not Acceptable
	- 408 ; Request Timeout
	- 409 ; Conflict
	- 429 ; Too Many Request
- 500번대
	- 500 ; Internal Server Error
	- 501 ; Not implemented - 서버가 지원하지 않는 함수 사용시.
	- 502 ; Bad Gateway
	- 503 ; Service Unavailable
	- 504 ; Gateway Timeout
